# 🎬 Movie Insights Dashboard – Power BI

## 📊 Overview
Interactive dashboard analyzing IMDb movie ratings, votes, and trends using Power BI.

## 🎯 Objectives
- Explore movie trends by duration, category, and release year
- Compare new vs old movies
- Identify top-rated and most-voted movies
- Enable data-driven insights for movie analysis

## 📈 Key Features
- Movie classification by genre, duration, and age rating
- Total ratings and votes per category
- Rating vs Duration analysis
- Comparison of old vs new movies
- Interactive table with clickable links to IMDb movies

## 🛠 Tools & Technologies
- Power BI
- DAX
- Data Visualization
- Data Analysis
- Interactive table links
